# Laboratorium 1 - konfiguracja środowiska



**Celem laboratorium jest opanowanie podstaw pracy z systemem kontroli wersji Git, platformą GitHub oraz przygotowanie lokalnego środowiska programistycznego dla wybranego frameworka – w tym przypadku Spring Boot z wykorzystaniem języka Kotlin. Szczególny nacisk położono na poprawne dokumentowanie pracy przy użyciu Markdown oraz konfigurację bezpiecznego dostępu do repozytorium za pomocą SSH.**

## Technologie:

* Kotlin
* Spring Boot
* Maven
* IntelliJ
* Java JDK 17 +

## Do uruchomienia projektu:

* Java JDK
* IntelliJ IDEA
* System Windows

## Uruchomienie aplikacji:

Uruchomienie projektu za pomocą polecenia:

`.\mvnw spring-boot:run`

Po uruchomieniu aplikacja będzie dostępna pod adresem:

`http://localhost:8080/api`
